# 공연 예매 사이트

---

공연을 등록 및 티켓 예매 사이트의 백엔드입니다.

## 개발 기간

---

- 23.12.28 - 24.01.02

### 개발 환경

- Framework : `NestJS`
- ORM : `typeorm`
- DB : `mysql`

## 주요 기능

---

**회원가입**
**로그인**
**프로필 보기**

**새 공연 등록(admin)**
**공연 검색하기**
**공연 상세보기**
**예매 확인하기**

**공연의 좌석 예매 정보 확인하기**
**좌석을 지정하여 예매하기**
**예매 취소하기**
